---作成物ファイル名-----------------
title = タイトル画面
char_select = キャラ選択画面
demo = 対戦画面
game_end = ゲーム終了後選択画面
------------------------------------


---前回からの主な修正点-------------
■title
　　読み込み時に少しフェードインされる
　　「スマホをセットしてください」の文字非表示バグ解消
　　　（Unityのデフォルトフォント以外のフォントを使用することで解決）


■char_select
　　「力士名」を変更
　　文字サイズ変更
　　力士選択・RIKISHI SELECT 位置修正
　　フェード時間修正


■demo
　　ゲーム終了画面を game_end に移動
　　選択された力士名を表示
　　勝敗決定時の演出追加
　　今後モーション「勝利」と「敗北」が入る（現在は、勝＝直立 , 負＝ダメージ　のモーションが設定されている）
　　フェード追加


■game_end
　　フェード追加

------------------------------------


---機能-----------------------------
■title
・Enterキー押下
　　一瞬フェードイン（白）
　　音声
　　フェードアウト（黒）
　　画面遷移 → char_select


■char_select
・↑キー / ↓キー押下
　　カーソル変更
　　効果音
　　力士名等変更

・Enterキー押下
　　モーション「勝利」（現在は直立モーションが設定されている）
　　フェードアウト（黒）
　　画面遷移 → demo


■demo　（選択されたキャラが渡されている）
・UnityのGame内で左クリック
　　「勝利」したときの画面表示
　　回転

・UnityのGame内で右クリック
　　「敗北」したときの画面表示
　　回転

・左右クリック後共通 Enterキー押下（一定時間経過後は強制的に以下が動く）
　　勝/敗 モーション設定
　　フェードアウト（黒）
　　画面遷移 → game_end


■game_end　（勝利 / 敗北が渡されている）
・↑キー / ↓キー押下
　　カーソル変更

・"次戦" / "再挑戦"にカーソルを合わせてEnterキー押下
　　フェードアウト（黒）
　　画面遷移 → demo

・"力士選択"にカーソルを合わせてEnterキー押下
　　フェードアウト（黒）
　　画面遷移 → char_select

・"終了"にカーソルを合わせてEnterキー押下
　　フェードアウト（黒）
　　画面遷移 → title
